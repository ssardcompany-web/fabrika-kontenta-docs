## 0. Технические рамки и правила платформ (must-read)

| Платформа | Видео-обложка (главное) | Видео внутри карточки | Соотношение | Звук | AR/AI ограничения |
|-----------|-------------------------|------------------------|-------------|------|-------------------|
| **Wildberries** | до 60 с (рекомендация 15–30 с), 720–1920 px, MP4/MOV ≤50 МБ | до 3 мин, до 1000 МБ, до 3 роликов | 9:16 для карточки, 3:4 (810×1080) для автозапуска | да (но многие смотрят без) | Лица модели не обрезать; манекены — только для шапок, белья, носков; запрещены логотипы конкурентов и WB-плашки «Wildberries рекомендует» |
| **Ozon (видеообложка)** | **8–30 с**, авто-loop, MP4/MOV, 1080p+, ≤20–30 МБ | до 5 мин, ≤100 МБ загрузка / 5 ГБ через YouTube | **3:4 для одежды/обуви/аксессуаров**, 1:1 для остального | **без звука** в обложке | Главное действие в первые 1–2 секунды; нужен бесшовный loop; для AI — рекомендация ≥30% реальных кадров (не блок, но влияет на ранжирование) |
| **Яндекс.Маркет** | 15 с – 2 мин (≤1 мин лучше), MP4 H.264 / MOV, ≤100 МБ | то же | **16:9** или **1:1** (вертикаль не приоритет) | да | 1280×720 минимум, рекомендуется 1920×1080 |
| **Mercado Libre (Clips)** | 10 с – 1 мин, MP4/MOV/MPEG/AVI, ≤280 МБ | через Clips и Mercado Libre Videos | **9:16 вертикаль** | требуется голос или фоновая музыка | Запрещены статичные кадры, чужой контент, прямые сравнения с конкурентами; нельзя выходить за safe-zone (кнопки share/fav) |
| **Lamoda** | **видео в карточке не поддерживается** (контент только фото) | n/a | n/a | n/a | Не работаем с видео |
| **Allegro** | до 60 с, MP4 H.264, ≤100 МБ, 720p+ | до 3 видео на карту | 16:9 или 9:16 | да | Без логотипов сторонних маркетплейсов |

### Целевая длительность шаблонов (наш стандарт):
- **8 с** — Ozon видеообложка минимум, loop-friendly, без звука
- **12 с** — универсальный «короткий хит» (WB обложка + Ozon обложка + Mercado кнопка)
- **15 с** — стандарт WB карточки + Mercado Clips
- **22 с** — расширенный сценарий с деталями (WB + Ozon + ЯМ)
- **30 с** — максимум для обложки Ozon, Mercado Clips, WB

### Универсальные правила всех шаблонов:
1. **Hook в первые 1–2 с** (75% решений «смотреть/листать» — в первые 2 секунды)
2. **Без распознаваемых лиц** — кадрировать выше глаз / в профиль / маска размытия (защита от бана WB и снижение AI-рисков)
3. **Без логотипов конкурентов и сторонних брендов** на одежде, фоне, аксессуарах
4. **Без текста поверх товара** в safe-zone (для Mercado — особенно правый-нижний угол)
5. **Loop-friendly финал** — последний кадр = первому ±200 мс (требование Ozon)
6. **Кириллический/латинский overlay** только в нейтральных зонах, шрифт ≥48 px на 1080p
7. **AI-сгенерированный фон/модель** допустимы, но кадры ткани/деталей — реальные (рекомендация Ozon)

### Формат 01 — Spin-360° на белом фоне («Видео-обложка 360°»)

| | |
|---|---|
| **EN** | 360° Turntable on White |
| **Категория** | универсал: платья, куртки, обувь, сумки, аксессуары |
| **Длительность** | **8 с** (loop) или 12 с |
| **AI-сложность** | **Easy** (turntable + ghost-mannequin / AI) |

**Beats (8 с loop):**
- 0.0–0.5 с — товар по центру, фронт
- 0.5–4.0 с — orbit 360° по часовой, постоянная скорость
- 4.0–7.5 с — продолжение / детали-zoom не нужны
- 7.5–8.0 с — возврат к фронту = первому кадру (бесшовный loop)

**Параметры шаблона:**
- Тип: товар отдельно (ghost-mannequin для одежды)
- Фон: белый студийный (#FFFFFF) или светло-серый (#F5F5F5)
- Камера: orbit статичная высота, скорость 45°/с
- Свет: рассеянный софтбокс 2 точки, без жёстких теней
- Модель: нет (ghost-mannequin) — обходит правило WB про лица
- Воспроизводимость: 100% (любой SKU)

**Правила:** WB ✓, Ozon ✓ (идеально для loop-обложки), ЯМ ✓, ML ✓, Allegro ✓
**Best для:** обувь, сумки, аксессуары, верхняя одежда. Базовый шаблон-обязателен для всех 1550 артикулов.
**Метрики:** +24% CTR обложки на Ozon (заявленные данные платформы).

### Формат 02 — Walk-on-model (модель идёт в кадре)

| | |
|---|---|
| **EN** | Fit-on-Model Walk |
| **Категория** | платья, верхняя одежда, костюмы, спортивные комплекты |
| **Длительность** | **12 с** или 15 с |
| **AI-сложность** | **Hard** (AI-движение тела + ткань-физика) |

**Beats (15 с):**
- 0.0–2.0 с — модель идёт фронтально к камере
- 2.0–5.0 с — медленный поворот вправо
- 5.0–8.0 с — профиль + взмах руки/плеча (показ драпировки)
- 8.0–11.0 с — поворот спиной (показ задней части)
- 11.0–14.0 с — возврат фронт + лёгкий smile/static
- 14.0–15.0 с — финальный фронт = первый кадр

**Параметры шаблона:**
- Тип: одежда на модели (полный рост)
- Фон: lifestyle (минималистичный — серый студийный, циклорама)
- Модель: skin tone {fair/medium/olive/dark}, body type {S/M/L/plus}, age {20–35} — параметризуется
- Камера: статика широкоугольник 24mm
- **Лицо:** обрезать выше уровня губ ИЛИ blur ИЛИ 3/4-back (правило WB)

**Правила:** WB ✓ (лицо вне кадра), Ozon ✓, ЯМ ✓, ML ✓ — НО для AI лучше реальная модель + AI-замена одежды (try-on)
**Best для:** платья, пальто, тренчи, костюмы — всё, где важна посадка в движении.
**Метрика:** конверсия в корзину на одежде с walk-видео — 25–35% против 15–18% без видео (рост 1.5–2×).
