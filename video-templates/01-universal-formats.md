# Универсальные fashion-форматы (до 30 сек)

> Дата: 2026-05-04
> Платформы: TikTok / Instagram Reels / YouTube Shorts (9:16, vertical)
> Контекст проекта: «Фабрика Контента» — AI-генерация продающих видео для брендов одежды через Veo 3.1 / Sora 2.
> Жёсткое ограничение: длительность ≤ 30 секунд (best engagement zone).

---

## TL;DR — что важно знать перед чтением

- **Окно решения зрителя — 1.7 сек.** Hook-overlay обязателен в кадре 0-2с.
- **3-second hold rate ≥ 60% выдаёт x5–x10 reach** (Instagram алгоритм 2026).
- **Best engagement window для fashion: 7–15 сек** (rapid transitions) и 15–30 сек (educational/styling). 60+ сек — для storytelling, не для нашего кейса.
- **TikTok apparel CTR-бенчмарк (paid): 0.69 %, ROAS 2.49, CPM $4.24, AOV $68.40.** Apparel — №2 по CTR среди всех индустрий.
- **Engagement rate органик fashion: 2.4 % avg, top brands > 8 %.**
- **Average watch time: 6–10 сек; > 50 % длительности = strong, > 70 % = viral candidate.**
- **AI-видео в 2026:** Veo 3.1 хорошо ведёт fabric/water/glass physics, есть character consistency через reference images; Sora 2 — лучший фотореализм, но без жёсткого character reference. Главные подвохи: микроэкспрессии лица, пальцы рук, мерцание текстур, drape ткани под нагрузкой.

## Детально по форматам

### 1. Outfit Transition / Смена образа

**Описание.** Один кадр — модель в образе А; быстрый swipe/clap/jump cut — модель в образе B (и далее C). Самый растиражированный fashion-формат TikTok.

**Длительность: 7–10 сек.** Обоснование: трансформация работает на сюрпризе — дольше 10 сек ритм рассыпается, теряется hook-effect. Бенчмарк HavStrategy: rapid transitions 7–15s дают +40 % engagement vs обычные fashion-Reels.

**Сценарий по beats:**
- 0–2s: hook — модель в casual look + текстовый overlay «Wait for it…» / «Этот образ за 1990 ₽»
- 2–4s: transition (clap / hand-swipe / 360 spin / fabric-flick)
- 4–8s: образ B raised — yamера фиксируется, модель держит позу 1.5 сек
- 8–10s: финальный кадр + лого/CTA «Свайп → корзина»

**Структурные элементы.**
- Сцен: 2–3
- Камера: статичный штатив (одна точка, чтобы trigger transition был чистым)
- Переходы: clap-cut / hand-swipe overlay / jump-cut по beat трека

**Параметры шаблона:**
- `product_type` — платье / костюм / комплект (top + bottom)
- `outfit_a` — базовый образ (casual)
- `outfit_b` — целевой образ (продаваемый)
- `location` — нейтральный фон (white wall / brick / mirror studio)
- `model_tone` — casual | glam | streetwear
- `audio` — beat-driven instrumental (drop в 0:02)
- `transition_trigger` — clap | swipe | jump
- `cta_text` — 5–7 слов, появляется в 0:08
- `brand_logo_corner` — TR / BR

**Best fit:** TikTok, Reels (vertical 9:16). Для WB-карточки — adapted версия 10–15s.

**Метрики.** Watch time avg ~ 7s (на 10s ролике = 70 % completion). Engagement rate 5–8 % у лидеров. CTR в био ~ 1.2 % (apparel TikTok organic).

**Примеры.** `@natalieoutfits`, `@chiaraferragni`, `@stylebydeb`, CapCut «1 Girl, 2 Outfits» template.

**AI-генерация: Easy.** Veo 3.1 справляется через 2 image prompts + character reference. Подвох — стык одной модели в обоих кадрах (face consistency); решается reference-image. Stop-motion-вариант проще, чем непрерывное движение.

### 2. GRWM (Get Ready With Me) / Сбор образа

**Описание.** Ускоренный монтаж процесса одевания: бельё → база → вторые слои → акценты → финал. Параллельно — войсовер или текстовые ремарки про вещь/повод.

**Длительность: 22–30 сек.** Обоснование: GRWM не работает на 7s — нужна последовательность ≥ 4 этапов. 22–30s — sweet spot, выше — падение completion.

**Сценарий по beats:**
- 0–3s: hook «GRWM на свидание в новом баре» + первый кадр (полотенце/халат)
- 3–10s: база — нижний слой / джинсы / водолазка (3–4 quick cuts)
- 10–18s: акцент — главная вещь (tops/dress) с замедлением
- 18–25s: аксессуары + макияж/причёска (skip-cuts)
- 25–30s: финальный full-body + CTA «Образ целиком в шапке»

**Структурные элементы.**
- Сцен: 6–10 micro-cuts
- Камера: handheld + штатив (микс), частичные крупные планы (руки, ткань, бирка)
- Переходы: jump-cut в ритм аудио, реже smooth fade

**Параметры шаблона:**
- `event_context` — свидание / работа / прогулка / вечер
- `hero_product` — главная вещь (платье / пиджак / boots)
- `supporting_items` — список 3–5 (джинсы, топ, аксессуары)
- `model_tone` — casual / glam / editorial
- `voiceover_script` — 30–40 слов, разговорный тон
- `audio_bg` — instrumental, 90–110 BPM
- `cta_text` — «Образ целиком в шапке», ссылка
- `vibe_tags` — `#grwm #ootd #вечеринка`

**Best fit:** Reels (educ./lifestyle), TikTok. Не подходит для marketplace-карточки.

**Метрики.** Save rate высокий (2–4× avg fashion). Avg watch time 14–18 сек на 30s. ER 3–6 %.

**Примеры.** `@brittanybathgate`, `@maizimmy`, `@matildadjerf`, тренд `#grwm` на TikTok = 47B views.

**AI-генерация: Hard.** Multi-shot continuity один персонаж, мимика, руки, fabric drape. Veo 3.1 с reference-image спасает на 60–70 %, остальное — montage из shorter generations + post-edit. Подвохи: пальцы при манипуляции одеждой, согласованность лица между cuts, физика ремня/застёжек.

### 3. Three-Fits-One-Week / 3 образа на неделю

**Описание.** «Понедельник / среда / пятница» — три образа подряд, каждый в своей локации/настроении.

**Длительность: 15 сек.** 3 образа × 5 сек = 15 сек. Меньше — не успевают зарегистрироваться, больше — теряется ритм.

**Сценарий по beats:**
- 0–2s: hook overlay «3 образа на неделю с одной кофтой»
- 2–7s: образ 1 (label «MON»), full-body + 1 деталь
- 7–12s: образ 2 (label «WED»), смена локации/освещения
- 12–15s: образ 3 (label «FRI») + CTA

**Структурные элементы.**
- Сцен: 3 (по 5s)
- Камера: статика на штативе, single locked shot per outfit
- Переходы: hard cut + lower-third label

**Параметры шаблона:**
- `core_item` — общий якорь (одна вещь во всех 3 образах) ИЛИ независимые
- `outfits[3]` — список с описанием каждого образа
- `location_scheme` — 1 локация × 3 света / 3 локации
- `day_labels` — `[MON, WED, FRI]` или `[Работа, Обед, Свидание]`
- `audio` — chill/hip-hop, drop на каждом образе
- `cta_text` — «Капсула в шапке»

**Best fit:** Reels, TikTok. Косвенно — для landing-страницы коллекции.

**Метрики.** Saves 2–3× avg (educational). Watch time 11–13s на 15s = 75–85 % completion.

**Примеры.** `@danielleberns`, `@songofstyle`, тренд `#3outfitsfor`.

**AI-генерация: Medium.** 3 разные сцены — face consistency через reference-image. Drape проблем мало (статика). Veo 3.1 предпочтительнее Sora 2.

### 4. Before/After Styling / 1 вещь, 3 образа

**Описание.** Одна и та же вещь (платье/пиджак/джинсы) показана в 3 разных стилизациях: casual, work, evening.

**Длительность: 15 сек.**

**Сценарий по beats:**
- 0–2s: hook — full-body в нейтральном виде + текст «Это платье можно носить 3 способами»
- 2–6s: способ 1 — Casual (белые кеды, джинсовка)
- 6–10s: способ 2 — Office (пиджак, лоферы)
- 10–14s: способ 3 — Evening (heels, серьги-кольца)
- 14–15s: CTA «Все аксессуары → линк»

**Структурные элементы.**
- Сцен: 4 (1 hero + 3 styling)
- Камера: статичный mirror selfie или 3/4 angle, штатив
- Переходы: outfit-swipe (поднял руку → следующий look)

**Параметры шаблона:**
- `hero_item` — платье/жакет/джинсы
- `style_variations[3]` — описание каждой стилизации
- `accessory_kit_per_look` — 3–4 предмета на образ
- `vibes` — `[casual, office, evening]`
- `text_overlay_per_scene` — 1 строка
- `audio` — upbeat pop instrumental

**Best fit:** Reels (educational), Pinterest cross-post.

**Метрики.** Save rate высокий (educational). Watch time ~ 11s/15s.

**Примеры.** `@ariellecharnas`, `@aimeesong`, hashtag `#3waystowear` (8B views).

**AI-генерация: Easy.** Один и тот же предмет одежды, 3 разных контекста — Veo 3.1 с product reference + character reference решает за один prompt-batch.

### 5. Try-On Haul / Примерка-подборка

**Описание.** Креатор показывает 3–5 вещей подряд, каждую — в полный рост, с микрокомментарием. Часто — после получения посылки/распаковки.

**Длительность: 22–30 сек.**

**Сценарий по beats:**
- 0–3s: hook — «Я заказала 5 вещей с {brand} — ставлю одной фразой»
- 3–8s: вещь 1 (5s: распаковка → primer на теле)
- 8–13s: вещь 2 + микро-вердикт «keep / send back»
- 13–18s: вещь 3
- 18–23s: вещь 4
- 23–28s: вещь 5
- 28–30s: CTA «Промокод HAUL10 → линк»

**Структурные элементы.**
- Сцен: 5 (по 5s)
- Камера: handheld для распаковки, штатив для try-on
- Переходы: jump-cut, иногда swipe-transition

**Параметры шаблона:**
- `brand_or_marketplace` — название
- `items[5]` — список с описанием, ценой, размером
- `verdict_per_item` — keep / return / undecided
- `audio` — voiceover-driven, instrumental BG quiet
- `discount_code` + `cta_text`
- `unboxing_intro` — yes/no (3s в начале)

**Best fit:** TikTok primary, Reels secondary. Идеально для marketplace-карточки коллекции.

**Метрики.** CTR в био 1.5–2.5 % (выше avg). Engagement 4–7 %. Conversion higher than transition (зритель уже в режиме «оценить»).

**Примеры.** `@kendalljenner` Aritzia haul, `@brittnimack`, `@daniellemarcan`, тренд `#tryonhaul` (54B views).

**AI-генерация: Medium.** 5 разных образов на одной модели — character consistency сложно держать на 30s; spliт на 5 generations + manual edit. Подвох: распаковка-сцена (руки + товар) часто ломается (Sora/Veo пальцы).

### 6. Fit-Check / OOTD

**Описание.** «Сегодня я в этом» — одиночный full-body shot текущего образа, 360°-поворот или статика-смена ракурса.

**Длительность: 7 сек.** Минимальная упаковка для алгоритма (3-second hook + 4 сек на образ).

**Сценарий по beats:**
- 0–2s: hook overlay «OOTD 14.05» / «Fit check»
- 2–5s: full-body 360° spin или static + crop angles
- 5–7s: CTA-overlay «Вещи → bio»

**Структурные элементы.**
- Сцен: 1
- Камера: штатив, fixed angle ИЛИ slow tracking pan
- Переходы: нет (single take) или fast crop-zooms

**Параметры шаблона:**
- `outfit_description` — 3–4 вещи
- `mood_tag` — `cozy | edgy | minimal | romantic`
- `weather_or_event` — context
- `audio` — short loop, 7s clip от трендового трека
- `cta_text` — `«Линк → магазин»`
- `caption_hashtags` — `#ootd #fitcheck`

**Best fit:** Reels, Shorts, TikTok. Daily content, низкая стоимость.

**Метрики.** View velocity отличная (low investment). Watch time 5.5–6s/7s = 80–85 %. Saves низкие.

**Примеры.** `@matildadjerf`, `@laurastoloff`, тренд `#ootd` (1T+ views).

**AI-генерация: Easy.** Одиночный shot, статика — идеальный кейс для AI. Veo 3.1 / Sora 2 оба справятся.

### 7. Stop-Motion Outfit Change / Покадровая смена

**Описание.** Кадр-за-кадром (~15 кадров): вещи «прыгают» на модель, аксессуары появляются по щелчку, образ собирается «магически».

**Длительность: 10–15 сек.**

**Сценарий по beats:**
- 0–2s: пустая модель в нижнем слое + hook «Watch the outfit build itself»
- 2–10s: stop-motion build (10–15 фреймов с интервалом 0.4–0.6s)
- 10–13s: финальный complete look, 360° или mirror
- 13–15s: CTA

**Структурные элементы.**
- Сцен: 1 локация, ~15 stop-frames
- Камера: locked tripod, абсолютно фикс
- Переходы: snap-cut (без crossfade)

**Параметры шаблона:**
- `final_outfit_description`
- `build_order` — порядок появления вещей (носки → джинсы → топ → пиджак → сумка)
- `frame_count` — 12 / 15 / 18
- `audio` — punchy beat (claps/snares совпадают с frame snaps)
- `pop_sfx` — yes/no
- `text_overlay_each_layer` — yes/no

**Best fit:** Reels, TikTok. Высокий share rate (visual-trick wow).

**Метрики.** Share rate 2× avg. Watch time 9–12s/15s. Saves выше среднего.

**Примеры.** `@rachelbarkules`, тренд `#stopmotionfashion` (480M views).

**AI-генерация: Hard.** Stop-motion требует строгой пиксельной согласованности модели между кадрами; AI image-to-image (Nano Banana / SDXL inpaint) подходит лучше, чем video-models. Сборка через ComfyUI workflow + ручной keyframing. Veo 3.1 stop-motion mode частично закрывает.

### 8. Mannequin Challenge / Статика → движение

**Описание.** Модель замерла в позе (как манекен), камера обходит вокруг → в кульминации модель «оживает» и двигается.

**Длительность: 7–10 сек.**

**Сценарий по beats:**
- 0–2s: hook — frozen pose + текст «Manneqiun mode»
- 2–6s: камера orbit вокруг модели (она не двигается)
- 6–8s: момент «оживания» — поворот головы / hair-flick
- 8–10s: модель идёт → CTA

**Структурные элементы.**
- Сцен: 1 непрерывная (orbit + reveal)
- Камера: handheld с гимбалом, smooth orbital motion
- Переходы: in-camera (без cut)

**Параметры шаблона:**
- `pose_description` — `power-stance | hand-on-hip | over-shoulder`
- `outfit_focus` — главная вещь
- `orbit_direction` — CW / CCW
- `awakening_action` — head-turn / hair-flick / smile
- `audio` — dramatic build → drop в 0:06
- `location` — studio / urban / nature

**Best fit:** TikTok, Reels.

**Метрики.** Completion 70–80 %. Engagement среднее.

**Примеры.** `#mannequinchallenge` (8B views), revival 2025–2026.

**AI-генерация: Medium.** Orbit camera move — Veo 3.1 умеет; «pose freeze» проще, чем walk. Подвох — переход от static к motion (момент «оживания»).

### 9. Walk-and-Twirl / Поворот в платье

**Описание.** Модель идёт навстречу камере или удаляется, делает поворот — платье/юбка раскрывается.

**Длительность: 7–10 сек.**

**Сценарий по beats:**
- 0–2s: hook — front-shot, начало движения + текст «Это платье просто Wow»
- 2–6s: walk towards / away
- 6–8s: twirl (slow-motion 4K@60fps recommended)
- 8–10s: CTA + brand-tag

**Структурные элементы.**
- Сцен: 1 непрерывная
- Камера: dolly-in / handheld follow или статика
- Переходы: speed-ramp (60fps → 24fps → slow-mo)

**Параметры шаблона:**
- `garment_type` — платье / юбка-миди / пальто / oversize-shirt
- `fabric_property` — flowy / structured / pleated
- `walk_speed` — slow-mo / normal / fast
- `location` — улица / studio / parquet / sand
- `lighting` — natural golden hour / hard shadow / soft daylight
- `audio` — orchestral / disco / r'n'b
- `cta_text`

**Best fit:** Reels (premium positioning), TikTok.

**Метрики.** Watch time высокий для premium-сегмента. ER 3–5 %. Brand recall сильный.

**Примеры.** Celine, Acne, `@matildadjerf`, `@chloecharm` walks.

**AI-генерация: Medium.** Walking + fabric flow — Veo 3.1 closest to camera footage по fabric physics. Подвох — feet/hands при walk-cycle. Sora 2 фотореалистичнее, но без жёсткого character-anchor.

### 10. Closet Pull / Достаём из шкафа

**Описание.** Рука тянется в шкаф / на штангу → выбирает вещь → надевает / прикладывает к себе. Часто — серия 3–5 пуллов.

**Длительность: 10–15 сек.**

**Сценарий по beats:**
- 0–2s: hook — крупный кадр шкафа + текст «Что надеть на дату #3?»
- 2–7s: выбор 1 (рука + вешалка + примерка)
- 7–11s: выбор 2 (отвергнут — «нет»)
- 11–14s: выбор 3 — финальный, full-body
- 14–15s: CTA

**Структурные элементы.**
- Сцен: 3–4 micro-scenes
- Камера: handheld или fixed POV
- Переходы: jump-cut + sound effect (вешалка-щелчок)

1. **Micro-Aesthetics over Macro-Trends.** Больше не «весь TikTok носит cargo» — сегментация на «Tech-Girl», «Glamoratti», «Poet-core», «Quiet Luxury Casual». Шаблоны должны иметь aesthetic-tag параметр.
2. **Maximalism Comeback.** Multi-layered tops, smushed colour-blocking, патчи и кастомизация. Visual «more-is-more» обыгрывается через rapid transitions / stop-motion build.
3. **Series-Driven Content.** Один формат → еженедельный сериал («Пятничный fit-check №14»). Алгоритм Reels/TikTok 2026 поощряет post-series.
4. **AR Try-On Integration.** Reels с AR-фильтрами для виртуальной примерки коллекции дают +30 % conversion. Шаблон Try-On Haul + AR overlay.
5. **Sustainability Narrative.** Thrift flip, cost-per-wear, capsule wardrobe — топ-форматы по share rate. Sustainability снова в маркетинговом топ-3.
6. **Audio-First Content.** Алгоритмы 2026 (особенно Reels) ранжируют по ratio «sound-on plays vs total». ASMR + trend-audio crucial. Silent video — penalty.
7. **Cost-Per-Wear Dissection.** Math-overlay в кадре — driver комментариев. Привязка цена-vs-носка превратилась в самостоятельный sub-genre (Y2026).

## Чёрный список: форматы, которые НЕ работают для чистого AI

| Формат                          | Почему                                                                                                          |
| ------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| **GRWM (#2)**                   | 6–10 micro-cuts одного персонажа + руки + мимика крупно. Face/hands consistency на 30s = текущий weak spot.     |
| **ASMR Fabric (#15)**           | AI-видео имеет frame-to-frame texture flicker, что ломает ASMR-эстетику и микро-движения ткани.                 |
| **Trend-Jacking (#13)**         | Тренды живут 7–14 дней, AI-pipeline отстаёт на 2–4 нед. Нужен hybrid: AI-обёртка + реальный creator.            |
| **Style Hack Tutorial (#19)**   | Hands-on close-up + поэтапные действия — пальцы + физика ткани крупно ломаются на 80 % кадров.                  |
| **Thrift Flip (#24)**           | DIY-процесс (резка, шитьё) crucial — AI ломается на инструментах + pinch-grip + физике разрезаемой ткани.       |

**Решение для Hard-форматов:** hybrid-pipeline. AI делает intro/outro/beauty-shot, реальный creator — критичный middle. Либо stop-motion-imitation вместо continuous motion.
