## Часть B — CAPTION TEMPLATES (Instagram Reels / TikTok / YouTube Shorts)

Длина оптимальная по платформам:
- IG Reels: 80–125 символов (короткое) ИЛИ 1500–2200 (длинное storytelling, "more" cut)
- TikTok: 100–150 символов (там caption < hook + sound)
- YT Shorts: 100 символов (вложен в title)

### B1. Шаблоны captions по формату (RU)

#### Outfit Transition / Fit-Check

- 3 brand hashtags (твой бренд + product line + collection)
- 3 niche hashtags (узкие — 10K–100K постов)

## Часть F — UTM TEMPLATES

### F1. Стандарт UTM для всех роликов

```
?utm_source={source}&utm_medium={medium}&utm_campaign={campaign}&utm_content={content}&utm_term={term}
```

| Параметр | Значения |
|---|---|
| `utm_source` | satellite / brand / bot / ads / direct |
| `utm_medium` | reels / tiktok / shorts / telegram / whatsapp / email |
| `utm_campaign` | имя сателлита или campaign-код (anya_kofty / fabrika_brand / L1_marketplace) |
| `utm_content` | видео-id (`v042` / `post23` / `msg7`) |
| `utm_term` | (опц.) ключевое слово для PPC |

### F2. Auto-builder UTM из YAML

```yaml
# template.yaml (фрагмент CTA)
cta:
  text: "{{ language.cta_marketplace_soft }}"
  destination_base: "https://t.me/fabrika_contenta_bot"
  utm:
    source: "{{ publisher.account_type }}"  # satellite / brand
    medium: "{{ publisher.platform }}"  # reels / tiktok / shorts
    campaign: "{{ publisher.account_handle }}"  # anya_kofty
    content: "{{ video.id }}"  # v042
  start_payload: "{{ segment }}_{{ template.variant }}_{{ utm.campaign }}_{{ utm.content }}"
  # → t.me/fabrika_contenta_bot?start=L1_A_anya_kofty_v042
```

## Сводная статистика

- **50 hook-паттернов** в 10 категориях × 2 языка = **100 готовых hook'ов**
- **20 caption-templates** по сегментам × 2 языка
- **80+ хэштегов** в банке (распределённые по 5 сегментам × 2 рынкам)
- **12 шрифтовых пар** + размеры + анимации (под 12 эстетик)
