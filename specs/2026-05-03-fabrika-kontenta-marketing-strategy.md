## 1. Контекст и наследство концепции

| Параметр | Из концепции |
|---|---|
| Продукт | Фабрика Контента — AI-генерация продающих видео для брендов одежды (8–60 сек для Reels / Shorts / TikTok / карточек МП) |
| Рынки | РФ + Аргентина одновременно. Универсальный продукт, локализация — верхний слой |
| ICP | 5 сегментов: маркетплейс-селлеры / DTC / ателье / швейные фабрики / крупные fashion-бренды |
| Pricing | Гибрид D — под каждый сегмент свой коридор: 49K ₽ / USD 199 → 490K+ ₽ / USD 1990 |
| Сайт | 7 концепций × 2 языка = 14 лендингов в A/B/C-тесте |
| Воронка целевая | от 0.3% до 1.3% сквозная (visit → paying) по бенчмарку SaaS B2B |

<a id="sec-2"></a>
## 2. Принцип product-led growth (продукт = маркетинг)

**Один из главных стержней стратегии:**

| Принцип | Что значит |
|---|---|
| **Продукт = маркетинг** | Каждый ролик в нашем feed одновременно (а) пример того, что продаём клиенту, (б) органический охват для алгоритма |
| **Маркетинг = product-output** | Production-команда роликов для клиентов = одновременно команда контент-маркетинга. Каждый клиентский ролик становится demo для следующих клиентов |
| **Стандарт качества feed = стандарт продукта** | Если feed халтура — никто не купит. Каждый собственный ролик = product-shipment |
| **Каждый клиент = новый ролик в нашей воронке** | Клиентская работа сразу = маркетинговый актив. Не нужна отдельная "креативная команда для рекламы" |

<a id="sec-3"></a>
## 3. RACE-фреймворк как backbone

| Этап | Что делает | Канал | Метрика |
|---|---|---|---|
| **R — Reach** | Awareness, "познакомить с проблемой и решением" | Сателлиты + бренд-каналы в Reels/Shorts/TikTok | Уникальные просмотры, охват/нед, подписчики |
| **A — Act** | Engagement, перевести смотрящего в активного | Лайки, комменты, follow, переход на лендинг | CTR из bio, % перехода на сайт |
| **C — Convert** | Конверсия в заявку/продажу | Лендинг + Telegram-бот | Conv. rate, CPL, CPQL, paying customers |
| **E — Engage** | Удержание + рост LTV + reference | Telegram nurture + клиент-loop | Retention, NPS, повторная покупка, кейсы для R |

### 5.4 Cross-pollination между сателлитами

| Тактика | Пример |
|---|---|
| Mention | "Аня" в Reels: "Слушала разбор у @anastasia.brand-strategist — попробовала её совет" |
| Дуэт / реакция | TikTok duet: "Светлана-ателье" реагирует на "@elena.fashion" lookbook |
| Recurring guest | "Михаил" иногда появляется как guest у "Аня" |

Эффект: smotr'итель видит **5+ разных людей**, каждый со своим путём, все говорят про одно → возражения снимаются через social proof.

<a id="sec-6"></a>
## 6. Stealth-engine: 6 слоёв правдоподобия

| Слой | Что делаем | Зачем |
|---|---|---|
| **1. Identity** | Уникальное имя без brand-патернов (`@anya_kofty89`, не `@anya_marketing`). Уникальный face-set (Veo + Nano Banana с consistent identity). Био без AI-меток | Не выглядит как marketing-аккаунт |
| **2. История** | Каждая персона = 3-летний story arc: где жила, как начинала, провалы и восстановления. Story bible = 3-страничный документ на персону | Внутренняя консистентность через посты |
| **3. Не-promo контент** | 70% постов = личное (день в жизни, переезд, новая собака). 20% = professional. **10% = natural mention продукта** | "Кажется реальным" — не "продаёт каждый пост" |
| **4. Imperfection** | Плохие кадры, оговорки в voice-over, опечатки в captions, "случайно" попавшие в кадр предметы | AI-content слишком идеален — несовершенство = маркер реальности |
| **5. Цифровой след** | Подписки персоны на других реальных людей, лайки на чужие посты, иногда комментарии. Реальные геолокации в постах | Профиль выглядит живым, не broadcast-machine |
| **6. Network behavior** | Cross-mentions с другими нашими персонами. Иногда упоминания реальных людей (engagement boost) | Видимость социальной ткани вокруг персоны |

<a id="sec-7"></a>
## 7. Production-pipeline на 15 сателлитов

| Роль | Кол-во чел | Работа |
|---|---|---|
| Story bible writer | 1 | 15 трёх-страничных историй один раз + апдейты |
| Сценарист RU | 1 | 20 скриптов/нед для 10 RU-персон |
| Сценарист ES-AR (носитель voseo) | 1 | 10 скриптов/нед для 5 AR-персон |
| Voice actor pool | 5–8 чел (контракт на голос с правом коммерч. использования) | Реальный voiceover для 70% контента |
| AI visual operator | 1–2 | Генерация face-set, motion, фон через Veo/Sora/Nano Banana/Midjourney |
| Editor | 2 | Финальная сборка ролика (color grade, sound design, captions) |
| SMM-оператор | 3 (по 5 персон каждый) | Постинг, ответы на DM/комменты, lite-engagement |
| Community manager | 1 | Cross-mentions, network behavior, реакции на инфоповоды |
| Quality auditor | 1 (час/день) | Проверка на uncanny valley, palpable promo |

**= ~10 человек / 30 часов в неделю каждый в первой фазе.** Оптимизация позже: editor + voice-actor pool можно переводить на freelance.

**Производственный темп:** 2 ролика/нед на персону × 15 = **30 роликов/нед**. Это ровно та мощность, которую продаём клиентам в тарифе Старт. Наш собственный production = 1 эквивалент Enterprise-клиента в нашу маркетинговую сеть.

<a id="sec-8"></a>
## 8. Обход platform detection

| Угроза | Mitigation |
|---|---|
| Одинаковый IP / VPN-провайдер | Пул из 30+ residential IP — каждой персоне свой статичный IP |
| Одинаковые device fingerprints | Виртуальные устройства с уникальным fingerprint (browser engine, screen resolution, time zone, accept-language) |
| Одинаковое время постинга | Случайный jitter ±90 мин, разные час-пояса (Москва / Питер / регионы; CABA / Cordoba / Rosario для AR) |
| Одинаковая стилистика монтажа | Свой пресет на каждую персону (color grade, шрифт subtitles, music palette, transition style) |
| Одинаковый promo-кэш | Промо-фразы про продукт переписаны на естественный voice каждой персоны |
| AI-detection алгоритмы платформ | Реальные voiceover на 70% + микро-imperfections + organic engagement |
| Coordinated inauthentic behavior detection | Network grows organically — не подписываемся массово в один день, не лайкаем массово свои же посты |
| Регистрационный паттерн | Разные email-домены, разные SIM (часть real numbers через nominal services), разные дни регистрации |
| Legal / DMCA risks | Только original content (face-set наш, музыка из rights-cleared банков), никакого деpfake реальных людей |

## 9. Cognitive Toolkit: 12 biases по этапам RACE

### Reach (видео в feed)

| Bias | Реализация |
|---|---|
| Mere exposure effect | Один зритель видит 5+ разных персон в неделю → доверие через повторение |
| Liking / similarity | Персона = его архетип ("такая же как я"). Видит "Аню селлера" — узнаёт себя |
| Authority bias | Сателлиты с success-историей ("оборот 3M на Ozon") = эксперт в его глазах |
| Bandwagon effect | Несколько персон в разных нишах используют один продукт = "все идут к этому" |
| Storytelling / narrative transportation | Hero's journey каждой персоны: до, переломный момент, после |
| Imperfect = real heuristic | Несовершенный контент парадоксально повышает доверие |

### Act (клик из bio на лендинг)

| Bias | Реализация |
|---|---|
| Curiosity gap | "В био — точная цена за 30 моих видео" → клик чтобы узнать |
| Reciprocity | Бесплатное демо в bio = персона "даёт ценность" → внутренний долг ответить |
| Commitment / micro-yes | "Кликни если вы такой же селлер" — крошечное обязательство |

### Convert (лендинг → заявка)

| Bias | Реализация |
|---|---|
| Anchoring | На лендинге — сначала альтернативная цена (фотосессия 100K) → потом наша (49K) |
| Loss aversion | "Ваш конкурент уже использует AI. Каждый день промедления — упущенные продажи" |
| Social proof | Логотипы / число клиентов / G2-rating-style звёзды / testimonials выше формы |
| Scarcity / FOMO | "Первые 5 клиентов в раннем доступе со скидкой −20% — осталось 2" |
| Risk reversal | "Не понравится — вернём деньги в течение 14 дней" |
| Future-self continuity | "Через 90 дней ваша карточка выглядит так →" + before/after мокап |
| IKEA effect | Калькулятор экономии — клиент сам считает, сам в это вовлекается |

### Engage (TG/WhatsApp nurture)

| Bias | Реализация |
|---|---|
| Sunk cost | "Вы уже скачали 3 наших гайда — попробуйте бесплатный пилот" |
| Endowment effect | Бесплатный пилот — клиент видит свой товар в нашем видео = ощущает что это уже его |
| Confirmation bias | Кейсы подтверждают "Я был прав, что это даёт результат" |
| Peak-end rule | Финальное сообщение в drip — самое сильное (live-эфир / personal outreach) |
| Halo effect | Одно "wow"-демо тянет за собой презумпцию качества всего |
| Identity-based | "Селлер, который не использует AI в 2026 — это селлер, который проиграет" |

<a id="sec-14"></a>
## 14. TG-бот компактный 4-дневный drip

Цель — снять все возражения и довести до покупки за 3-4 дня, пока лид горячий, не размазывать на месяц.

| Время | Сообщение | Cognitive bias | Цель |
|---|---|---|---|
| **Day 0, мгновенно** | Лид-магнит (бесплатное демо его артикула) + welcome от бота | Reciprocity | Зацепить wow-эффектом |
| **Day 0, +2 часа** | "Видели результат? Вот 5 быстрых FAQ от других селлеров" | Curiosity gap + authority | Снять первые возражения |
| **Day 1, утро** | Кейс из его сегмента с цифрами (CTR +40%, ROAS×2) | Social proof + similarity | Доказать что даёт результат у похожих |
| **Day 1, вечер** | Сравнение "стоимость текущего контента vs наша подписка" — калькулятор | Anchoring + IKEA | Показать экономику |
| **Day 2, утро** | Ответ на главное возражение его сегмента (для МП — "что если WB не пропустит"; для DTC — "стиль не уйдёт?") | Risk reversal | Снять блокирующее сомнение |
| **Day 2, вечер** | Разбор второго возражения + бесплатный пилот / discovery call | Reciprocity + commitment | Ввести в micro-yes |
| **Day 3, утро** | "Limited offer: −20% на первый месяц если оплатите за 24 часа" | Scarcity + loss aversion | Forcing function |
| **Day 3, вечер** | Личное сообщение от Александра (не бот) — "вижу вы смотрели наши кейсы, что не так?" | Personal touch + curiosity | Прорыв через личный канал |
| **Day 4, утро** | Финал: "Цена возвращается к стандартной через 4 часа, последний раз" | Loss aversion + scarcity | Last chance |

После Day 4 — режим переключается на пассивный: лид остаётся в канале `@fabrika_contenta` (RU) / `@fabrica_contenido` (ES) и получает регулярные посты по 7 типам контента (8/неделю). Mere exposure через органику.

Если за 3-4 дня не сконвертился → попадает в pixel-ретаргет + получает 1 email "что вы думаете о нашем сервисе?" (single-touch, не серия).

<a id="sec-16"></a>
## 16. Социальный proof как стержень

> Принцип: на каждом этапе RACE, в каждом канале, в каждом touch-point — доминирует не "наше обещание", а "доказательство от других". Возражения не снимаем аргументами — снимаем демонстрацией.

### 16.1 Семь типов social proof

| Тип | Что это | Где силён |
|---|---|---|
| 1. Customer logos / brand wall | Стена логотипов клиентов или "as seen in / used by" | Лендинг hero, бренд-канал bio |
| 2. Number proof | "X клиентов / Y роликов / Z брендов / N% рост" | Hero лендинга, посты в TG |
| 3. Case study с цифрами | Конкретный клиент → конкретный рост | Лендинг proof-block, посты в TG, видео-кейсы |
| 4. Testimonial цитата | Прямая цитата клиента (текст или видео) | Лендинг + TG + WhatsApp |
| 5. UGC | Когда клиенты сами упоминают нас в соц-сетях | Repost в наш бренд-канал |
| 6. Influencer / peer endorsement | Блогер / эксперт ниши хвалит | Reels/Shorts от сателлитов |
| 7. Live demonstration | Эфир, реальный разбор клиента в realtime | Telegram Live, IG Live |

### 16.2 Матрица: какой proof на каком этапе RACE

| Этап | Доминирующий тип proof | Почему |
|---|---|---|
| R — Reach | #6 Peer endorsement (через сателлиты) + #5 UGC | "Снять возражения у незнакомца через признание авторитетов" |
| A — Act | #1 Logos + #2 Numbers в Hero | "За 3 секунды убедить что мы не самозванцы" |
| C — Convert | #3 Case study + #4 Testimonial выше формы | "Снять последнее возражение прямо перед формой" |
| E — Engage | #3 Case study еженедельно + #7 Live demo ежемесячно | "Поддерживать proof-поток для не-сконвертившихся" |

### 16.3 Производственный конвейер proof в три фазы

| Фаза | Когда применяем | Источник proof |
|---|---|---|
| **Phase 0 — Synthetic proof** | 0–50 клиентов | Сателлиты = 15 AI-персон рассказывают как Фабрика изменила работу. Демо-видео с числами "до/после" |
| **Phase 1 — Pilot proof** | 5–30 клиентов | Договариваемся с первыми 5–10 клиентами на бесплатный пилот в обмен на разрешение использовать кейсы |
| **Phase 2 — Self-fueled proof** | 30+ клиентов | Каждый новый клиент = новый кейс по контракту (стандартный B2B opt-in) |

### 16.4 Один кейс — 5–7 touch-points

```
Реальный кейс клиента (например — селлер увеличил CTR на 40%)
    │
    ├─→ Видео-разбор → ролик на @fabrika.contenta (Reels/Shorts/TikTok) [Reach]
    │
    ├─→ Цитата клиента → Hero L1 / L2 (лендинг) [Act + Convert]
    │
    ├─→ Развёрнутый текст → пост в Telegram-канале (RU + ES) [Engage]
    │
    ├─→ WhatsApp broadcast (ES + EN) на список подписавшихся [Engage]
    │
    └─→ Сателлит "Аня, селлер кофт на WB" делает natural mention в Reels [Reach + Engage loop]
```

## 17. Act-этап детально

### 17.2 Quick attribution через UTM

| Источник | UTM-параметры |
|---|---|
| Сателлит "Аня, селлер кофт" → L1 | `?utm_source=satellite&utm_medium=reels&utm_campaign=anya_kofty&utm_content=video_NN` |

## 18. Convert-этап детально

### 18.1 Детальная воронка одного лендинга

```
Visit (100%)
  │
  ▼
Видит Hero (97%) ← bounce 3% сразу
  │
  ▼
Скроллит к pricing (45–60%)
  │
  ▼
Доходит до формы (15–25%)
  │
  ▼
Открывает форму (8–12%)
  │
  ▼
Заполняет 3 поля (4–7%)  ← основная метрика CR
  │
  ▼
Отправляет заявку (3.5–6%)
  │
  ├── Авто-ответ в TG/WA: "Получили, ответим в 24 ч + бесплатное демо"
  │
  ▼
Менеджер берёт лид (24 часа SLA)
  │
  ├── Квалификация по 3 критериям (бюджет, сегмент, готовность)
  │
  ▼
Discovery call (40-60% from qualified leads)
  │
  ▼
Договор (25-40% from calls)
  │
  ▼
Первый платёж
```

**Бенчмарки** (из ресёрча Unbounce + типичные SaaS B2B):
- Visit → Form submit: 3.8% (median) / 11.6% (top 25%)
- Form submit → Qualified lead: 40-60%
- Qualified lead → Discovery call: 40-60%
- Call → Closed deal: 20-40%
- Сквозная воронка: **0.3% – 1.3%** от visit к paying customer

### 18.2 SLA по обработке лида

| Этап | Целевое время | Кто отвечает |
|---|---|---|
| Авто-ответ в TG/WA после заявки | <30 сек | Bot |
| Первый человеческий ответ | <2 часа в рабочее время / <24 часа в нерабочее | Sales-менеджер |
| Discovery call назначен | <48 часов | Sales-менеджер |
| Бесплатное демо-видео отправлено | <24 часа после получения артикула | Production-team |
| Коммерческое предложение | <72 часа после discovery | Sales-менеджер |

### 18.3 Quality Score лида (для отсева)

Вход в pipeline только если **сумма баллов ≥ 5** из 10:

| Критерий | Баллы |
|---|---|
| Бюджет на маркетинг указан или подтверждён | 0/3 |
| Целевой сегмент (5 ICP) совпадает | 0/2 |
| Срочность (хочет в ближайшие 30 дней) | 0/2 |
| Decision-maker (не junior) | 0/2 |
| Реальный бизнес (есть IG/сайт/магазин) | 0/1 |

Лиды <5 баллов → не на discovery call, а в TG/WA nurture с label "low_priority".

## 19. Engage-этап детально

<a id="sec-21"></a>
## 21. Runner-протокол первых 90 дней

3 спринта × 30 дней. Цель — за 90 дней пройти от 0 до first 5 paying customers + готовая proof-инфраструктура.

### Спринт 1 (День 0–30): Foundation

| Что делаем | Кто | Конкретно |
|---|---|---|
| Регистрация всех аккаунтов | SMM-команда | 15 IG + 15 TikTok + 10 YouTube + 4 nurture-канала (TG×2 + WA×2) — всего 44+ аккаунта. Разные IP, fingerprints |
| Story bibles на 15 персон | Story writer | 15 × 3-страничный документ |
| Production-команда сформирована | Founder | 10 человек по ролям из раздела 7 |
| 14 лендингов задеплоены (без копирайта пока) | Frontend dev | Skeleton по 7 концепциям × 2 языка |
| GA4 + UTM + Pixels настроены | Аналитик | Tracking всех конверсий |
| Telegram-бот функционирует по сценарию L1-L7 | Backend dev | Маршрутизация по UTM |
| WhatsApp Business каталог настроен | SMM-команда | ES + EN catalog |
| Stack monitoring настроен | Аналитик | Дашборд Reach/Act/Convert/Engage в одном экране |

**Метрика конца спринта 1:** 44+ аккаунтов активны, 14 лендингов отвечают HTTP 200, бот принимает заявки, дашборд собирает данные.

### Спринт 2 (День 31–60): Production at scale

| Что делаем | Кто | Конкретно |
|---|---|---|

## 23. Что НЕ входит в стратегию (anti-scope)

- **Реализация сайта** (HTML/CSS/JS, фреймворк, хостинг, CMS) — Sub-project 3
- **Платформа продукта** (workflow генерации, бэкенд, очередь Veo/Sora, доставка) — отдельный технический спек
- **Юр. документы** (договор оферты RU + cross-border USD-договор + opt-in пункты) — отдельный legal-спек
- **Финансовая модель** маркетинга (бюджет на промоушен, P&L канала, CAC по каналам) — пока зафиксирована только канальная карта; конкретный бюджет — отдельный финансовый артефакт после первых данных пилота
- **HR и найм production-команды** — отдельный HR-план
- **Бренд-айдентика** (логотип, гайдлайн, type-system) — отдельный design-спек
- **Influencer-маркетинг с реальными людьми** (партнёрки с реальными блогерами) — после стабилизации сателлитов, отдельная инициатива
- **Расширение в Mexico/Chile/Colombia** — после стабилизации AR
- **PR / СМИ** — после первых 30 paying customers
- **Конкретные тексты** drip-сообщений, скриптов сателлитов, копирайт лендингов — переходит в Sub-project 3
